# Kareem Muftee — Portfolio

Single-page portfolio with immersive HTML Canvas physics effects.

## Stack

- **React 18** + **TypeScript** + **Vite**
- **Tailwind CSS** v3
- **Framer Motion** v11
- HTML Canvas (all physics/creative sections)

## Sections

| Section | Canvas Effect |
|---------|--------------|
| Hero | 3D Fibonacci sphere from name characters, mouse drag + inertia |
| About | Text ripple field — mouse creates wave through background characters |
| Skills | Verlet rope physics — keywords dangle under gravity, mouse magnet |
| Work | Project cards with obstacle-flow text background |
| Contact | Galaxy text vortex with trail effect, mouse-shifted center |

## Run locally

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173).

## Build

```bash
npm run build
npm run preview
```
